# 📊 Cálculo de Média em Python

## 📌 Sobre o projeto
Este projeto foi desenvolvido durante meus estudos em Python, com o objetivo de praticar lógica de programação e validação de dados.

---

## ⚙️ Funcionalidades
- Entrada de 3 notas
- Validação (valores entre 0 e 10)
- Cálculo da média
- Verificação de aprovação ou reprovação

---

## 🧠 Conceitos aplicados
- Estruturas de repetição (`while`)
- Condicionais (`if/else`)
- Validação de dados
